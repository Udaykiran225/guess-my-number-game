# Guess My Number Game 🎮

An interactive Android number guessing game developed using Kotlin in Android Studio.

## Features
- Random number generation
- User-friendly interface
- Score tracking system
- Input validation
- Responsive XML-based UI design
- Smooth gameplay experience

## Tech Stack
- Kotlin
- Android Studio
- XML Layouts

## What I Learned
- Android app development fundamentals
- Kotlin programming concepts
- UI/UX design using XML
- Event handling and app logic implementation
- Debugging and problem-solving in Android projects
