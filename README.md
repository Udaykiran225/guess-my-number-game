рџЋ® How the Game Works

1. The application generates a random number.
2. The player enters a guess.
3. The app checks the entered number.
4. Hints are displayed:
    * Too High
    * Too Low
    * Correct Guess
5. The game continues until the correct number is guessed.
6. User can restart the game anytime.
